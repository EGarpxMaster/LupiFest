# Directorio de Imágenes

## Instrucciones

Coloca tus fotos en este directorio siguiendo esta estructura:

### Miniaturas (Thumbnails)
Carpeta: `thumbs/`
- Tamaño recomendado: 400x300px
- Formato: JPG o PNG
- Nombres: 01.jpg, 02.jpg, 03.jpg, etc.

### Imágenes Completas (Fulls)
Carpeta: `fulls/`
- Tamaño recomendado: 1200x900px o mayor
- Formato: JPG o PNG
- Nombres: 01.jpg, 02.jpg, 03.jpg, etc.

## Ejemplo de estructura:

```
images/
├── thumbs/
│   ├── 01.jpg
│   ├── 02.jpg
│   ├── 03.jpg
│   ├── 04.jpg
│   ├── 05.jpg
│   └── 06.jpg
└── fulls/
    ├── 01.jpg
    ├── 02.jpg
    ├── 03.jpg
    ├── 04.jpg
    ├── 05.jpg
    └── 06.jpg
```

**Nota**: Asegúrate de que cada miniatura tenga su correspondiente imagen completa con el mismo número.
